# Latin Syntax Analyzer

Analyzes *Latin sentences* (1st and 2nd decl., verbs in pres.; more categories shall be added in the future), finding verb, subject and object as well as identifying categories such as declension, case, etc. and tense, person, etc.

Skills: Lists, Dictionaries, Modules.

## Motivation

Program created as Hackbright Final Project.

First step to a translator based on syntax, not semantics.
